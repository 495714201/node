这里面可以设置全局安装的目录，可以设置代理


文件模块调用，require("./math.js") ，需写上路径。
核心模块和第三方模块调用，require("mine")。直接写名字就可以，不用写路径。
不是路径的情况，node会先去找是否是核心模块，即是否是自己亲生的那个模块，如果没有，就会往node_module里去找（或者说一级一级往上找，一直到顶级找到为止），如果不能找到，就会报错。
